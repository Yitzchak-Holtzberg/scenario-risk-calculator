v0.6 layout candidates

Three different takes on showing the new two-tier outcome — pick one to ship.
The data we're laying out: per country, two probabilities (mobilization + armed violence/coup) each with a 90% CI, plus 10 raw feature values each with two per-tier logit contributions. The current dashboard tries to fit it all into a single vertical layout; these three take different approaches to using horizontal space.
Option 1 · Spatial

Atlas

A scatter plot of all 158 countries on a (mobilization, violence) plane dominates the screen. Click any dot to drill into a country card. Lets you see where each country sits relative to the rest of the world at a glance — four natural quadrants (stable / mass-mobilization-only / armed-only / both).

Best for: "show me the global picture, then let me dive into one country."
Option 2 · Editorial

Brief

A magazine-style country brief. Big country masthead, twin probability blocks with CI bars side by side as the hero, then a long evidence table showing each feature's per-tier logit contribution as parallel bars (one push-left/right per tier). Sliders + reference class + methodology in a sticky sidebar.

Best for: "give me everything I should know about this one country, formatted like a research brief."
Option 3 · Tabular

Index

A sortable, filterable table of every country with twin probability bars (mob | violence) per row. Search by name, filter by regime, sort by either tier. Click any row to expand inline into a detail view with per-feature contribution columns + scenario sliders. Densest information per screen.

Best for: "show me the rankings, let me scan and compare, then drill in only when something stands out."
